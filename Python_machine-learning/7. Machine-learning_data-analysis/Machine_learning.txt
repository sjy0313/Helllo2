규칙기반 방식 : 어떤 작업을 시키기 위해서 프로그램을 작성해서 지시하는 방식
기계학습 방식 : 데이터를 기반으로 컴퓨터가 스스로 학습을 하여 문제를 해결하는 방식
BRMS(Business Rule Management System)조직/기업을 운영하는데 사용되는 결정 논리의 정의/설치/수행감시 그리고 유지보수하는 소프트웨어
SES(system experts system)전문가가 갖고 있는 전문지식을 인위적으로 컴퓨터에게 부여하여 상호대화를 통해 
원하는 결과를 얻는 일종의 자문형 컴퓨터 시스템
지도학습(회귀분석/ 분류) 정답을 알고 있는 상태에서 학습(모형평가 방법이 다양)
<-> 비지도학습(군집분석) 정답이 없는 상태에서 서로 비슷한 데이터를 찾아서 그룹화 (모형 평가 방법이 제한적)
 
 
 
기계학습의 큰 분야(강화/지도/자율 학습) 
강화학습 : 보상 및 처벌의 형태로 학습 데이터가 주어진다 주로 차량운전/ 상대방과의 경기 같은 동적인 환경에서 프로그램의 행동에 대한 피드백만 제공되는 경우
 Agent -> environment ( action <-> reward )
지도학습 : 문제와 정답을 알려주고 학습을 시킨다. 입력-출력 쌍으로 학습 후 새로운 값이 들어왔을 떄 합리적인 출력값(f(x)=y) 를 예측하는 것
즉, 입력(X). 출력(y)가 주어질 떄 f(x)를 찾는 것


사이킷런 모듈은 회귀 모형에 기반한 기계학습을 쉽게 구현할 수 있도록 해 준다. 
데이터를 분류하는 알고리즘의 대표 : k-NN알고리즘
기계 학습을 통해 만들어진 모델은 과적합이나 과소적합을 보일 수 있다
과적합 시에는 데이터를 늘이고 , 과소적합에는 특징을 늘려볼 수 있다. 
회귀분석(Regression)  : 연속변수를 예측하는데 주로 사용(가격/매출/주가/환율/수량)
분석모형이 예측하고자 하는 목표를 종속(dependent)변수 or 예측(predictor)변수
예측을 위해 모형이 사용하는 속성을 독립(indenpendent)변수 or 실명(explanatory)
(예시 :소득이 증가(x) -> 소비가 증가(y) x변수가 y변수에 영향을 줄떄 두변수 사이에 선형관계가 있음)
일대일로 대응되는 확률적, 통계적 상관성을 찾는 알고리즘을 단순회귀분석이라고 함(대표적인 지도학습 유형)
x,y 에 대한 정보를 가지고 y축과 교차하는 지점이 y절편을 반복학습을 통해 찾는다. 일차방적식의 계수a,b를 찾는 과정(y=ax + b)
데이터를 봤을 떄 어떤 규칙성이 있어야 인공지능을 활용가능 즉, 데이터 분석단계에서 상관관계를 파악한 후에 인공지능의 활용여부를 판단해야 한다. 

어떤 상황에서 어떤 모델을 쓸지를 학습하기 위해 모델 별 기능 및 어떤 데이터가 어떤 모델에 적합한지를 판단하는 것이 중요
