테마 :
스타일 테마 설정 (5가지: darkgrid, whitegrid, dark, white, ticks)


그래프 :
선 그래프 (Line Plot): 데이터 포인트를 선으로 연결하여 시계열 데이터나 연속적인 데이터의 변화를 표현합니다.
산점도 그래프 (Scatter Plot): 데이터 포인트를 점으로 표현하여 변수들 간의 상관 관계나 패턴을 시각화합니다.
막대 그래프 (Bar Plot): 범주형 데이터의 각 범주에 대한 빈도수, 비율 등을 막대로 표현합니다.
히스토그램 (Histogram): 연속형 변수의 분포를 막대로 나타냅니다. 각 막대는 변수의 구간을 나타내며 빈도수를 표시합니다.
원 그래프 (Pie Chart): 범주형 데이터의 각 범주가 전체에서 차지하는 비율을 원형으로 나타냅니다.
상자 그림 (Box Plot): 연속형 변수의 요약 통계량인 최솟값, 제1사분위수, 중앙값, 제3사분위수, 최댓값 등을 사용하여 데이터의 분포와 이상치를 시각화합니다.
히트맵 (Heatmap): 2차원 데이터를 색상으로 나타내어 데이터 간의 관계를 시각화합니다. 주로 상관 행렬이나 빈도표를 표현하는 데 사용됩니다.
등고선 그래프 (Contour Plot): 3차원 데이터의 등고선을 그림으로 나타냅니다. 등고선은 같은 값을 가지는 지점들을 연결한 선으로 이루어져 있습니다.
3D 플롯 (3D Plot): 3차원 데이터를 시각화합니다. 주로 scatter plot, surface plot 등을 그릴 때 사용됩니다.
파이 차트 (Pie Chart): 범주형 데이터의 상대적인 비율을 부채꼴 형태의 그래프로 나타냅니다.
포함 플롯 (Inset Plot): 큰 그래프 내부에 작은 그래프를 포함하여 상세한 정보를 보여줄 때 사용됩니다.
