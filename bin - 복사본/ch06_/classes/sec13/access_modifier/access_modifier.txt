접근 제한자(Access Modifier)

접근제한자		제한대상						제한범위				설명
public			클래스/필드/생성자/메소드 		없음					모든 패키지에서 접근 가능
protected		필드/생성자/메소드				같은 패키지, 자식객체	상속된 자식 개체 
(default)		클래스/필드/생성자/메소드		같은 패키지				명시적으로 지정하지 않음 .
private 		필드/생성자/메소드				객체 내부				다른 패키지에서는 접근 불가능 해당 클래스에서만 접근 가능


1. class 에는 protected private 를 지정할 수 없다
2. protected private 은 필드/생성자/메소드에만 지정할 수 없다
