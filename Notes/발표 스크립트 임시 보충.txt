대주제 : 국내인구이동통계를 이용한 빅데이터 분석(수정 가능)
최종 목표 : 국내인구이동통계를 이용한 미래의 부동산 가격 예측(수정 가능)
1. 배경 :

	- 주제
	- 주제 선정 이유 : 저출산과 더불어 한국에서 매우 강하게
	나타나는 특징이 바로 수도권 밀집 현상입니다. 현재 대한민국 인구 절반 이상이 수도권(경기,서울,인천)
	에 거주 중입니다. 데이터를 바탕으로 현재 대한민국의 인구이동 현황은 어떻게 되는지, 
	왜 이러한 현상이 발생하는지, 이유는 무엇인지 등등을 분석해보고 이를 바탕으로 국내인구이동통계를 이용한
	미래의 부동산 가격을 예측해보려 합니다.(수정 가능)
	
	

2. 분석 계획 :

	- 수집 대상 데이터 선정 : 통계청(KOSIS)에 있는 국내인구이동통계 데이터(각 개인별 역할에 따라 추가 가능)
	
	- 역할 소개(개인별로 확정시 추가)
	    - 신정윤 : 
		- 성우진 : 
		- 오승필 : 
		- 박주찬 :
		- 이의재 : 국내인구이동통계 데이터 전처리
		
3. 분석 과정 :

	1) 데이터 수집 
		- 대통령실 홈페이지 연설문 웹크롤링
		
		- 데이터셋 : 문민[국민의]정부 이후의 모든 연설문
			- 목표 데이터셋 : 역대 대통령들의 모든 연설문
			- 축소 이유 : 
					1. 크롤링 함수 최적화 문제 :
					2. 시대적 상황의 문제 : 문민[국민의] 정부 이전은 반공 정서로 인해 과격한 단어 사용이 잦아 결과의 신뢰성이 훼손될 수 있음.
					[3. 시간 문제 : 데이터 수집 및 전치리 시간이 오래 걸리는데 비해 프로젝트 기간이 짧음]
		
		- 툴 : request 패키지, BeautifulSoup 패키지
		
		- 웹크롤링 함수
			- 입력 : 연설문 게시판 내부 idn(artid)
			- 출력 : 대통령 연설문 str이 저장된 리스트
		
		
	2) 데이터 정제(전처리)
	
		- 형태소 분석하여 명사 추출
		- 불용어 설정 및 제외
		
		- 툴 : 
			konlpy 패키지의 Komoran 모듈 : 한국어 형태소 분석기
			collections 패키지의 Counter 모듈 : 단어 빈도 카운팅
			re 모듈 : 불용어 처리
			
		- 전처리 함수
			- 입력 : 대통령 연설문 str이 저장된 excel 파일
			- 출력 : dataFrame 객체 (단어별 출현 빈도)
		
	3) 데이터 시각화 
	
		- Word Cloud 생성 모듈 작성
		- 막대 그래프 생성 모듈 작성
			- 입력 : dataFrame 객체 (단어별 출현 빈도)
			- 출력 : 그래프 출력, 이미지 저장
			
		- 툴 : 
			matplotlib : 그래프 시각화, 막대 그래프 생성,
			wordcloud : 워드클라우드 제작 라이브러리
			PIL : (워드클라우드)이미지 처리 라이브러리
		
	4) 백엔드
		- 툴 : streamlit 패키지
	
4. 결론

	   국제적인 지원을 받기 위해 주변 국가들과의 긴장 관계를 유지하고자 하는 경우, 
	   국내 안보를 강화하거나 주민들의 지지를 확보하는 데에도 사용됩니다. 즉 여러 가지 이유가 복합적으로 적용됩니다
	   따라서 연설문 분석 만으로 북한과의 대북 갈등 원인, 대남 도발 이유를 정확하게 파악하기는 어렵습니다]

			[- 데이터 수집 및 전처리 시 소요 시간이 오래 걸려서 역대 대통령들의 모든 연설문을 크롤링하는데 실패]

			- 변형 워드클라우드의 마스킹 이미지 수집의 어려움.
