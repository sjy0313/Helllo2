주제 선정
1. 최종 목표 : 국내인구이동통계 데이터를 바탕으로 한 미래의 부동산 가격 예측?
2. 그렇다면 데이터를 바탕으로 바로 분석? 아니면 5명이 각자 정한 소주제 분석 후 그것을 바탕으로?
3. 만약 소주제 분석 후 그것을 바탕으로 한다면 소주제 목록(토요일 디스코드 회의로 결정)은
다음과 같습니다
- 신정윤 : 연령이 인구 이동에 미치는 영향
- 성우진 : 국내인구이동통계를 바탕으로 한 정책 제안
- 이의재 : 출생률과 지역적 특성
- 박주찬 : 교육 수준과 인구이동의 연관성
- 오승필 : 1인 가구를 바탕으로 분석, 외국인 전출 전입 데이터를 바탕으로 분석
4. 국내인구이동통계 데이터 분석 결과
서울특별시에서 경기도로 이동한 인구 비율이 42% 초과, 압도적인 비율
그렇다면 만약 각자 분석 시 승필이형이 제안한
- 서울특별시 -> 경기도 이동 데이터는 경기도를 각 시로 분할 후 분석
- 비수도권 -> 비수도권 데이터는 제외
- 수도권(경기도,서울특별시,인천광역시)가 하나라도 포함된 데이터만 분석
이 기준으로 통합해서 분석할 것인지, 아니면 각자의 기준을 가지고 분석할지 결정이 필요할 것 같습니다
5. 기타 역할분담
현재 의재형과 승필이형이 국내인구이동통계 데이터 전처리를 완료한 상태입니다.
앞으로 역할 분담할 때 참고하면 좋을 것 같습니다
